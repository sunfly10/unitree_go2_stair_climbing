# unitree_go2_stair_climbing

### 1. 서론

본 프로젝트는 포스텍(POSTECH) 연구참여 프로그램을 통해 진행한 사족보행 로봇(Unitree Go2) 제어 실습 프로젝트입니다. 

발전소 등 복잡한 필드 환경에서 로봇이 계단을 자율적으로 인식하고 등반하기 위한 전 단계인 '정밀 접근 및 시선 제어 시스템'을 구축했습니다.

기존 3D 센서 퓨전(Camera-LiDAR) 시 발생하는 보행 진동 노이즈 한계를 극복하기 위해, 화면상의 픽셀 정보를 직접 제어 입력으로 사용하는 IBVS(Image-Based Visual Servoing) 방식을 도입한 것이 핵심입니다.

•개발 환경: Ubuntu 22.04, ROS 2 (foxy), Cyclone DDS 

•하드웨어: Unitree Go2, Intel RealSense D435i

•핵심 기술: YOLOv8 객체 인식, IBVS 기반 모션 제어, 적응형 임계값 알고리즘, 기구학 기반 피치(Pitch) 제어

### 2. 주요 기술 및 알고리즘

•환경 적응형 비전 인식 (YOLOv8)

데이터셋 최적화: Roboflow를 활용하여 계단 전체 영역(stair)과 개별 층계(step)를 세분화하여 학습시켰습니다. 이를 통해 로봇 앞의 장애물이나 사람에 의해 계단 일부가 가려지는 상황(Occlusion)에서도 인식의 연속성을 확보했습니다.

적응형 임계값(Adaptive Thresholding): 로봇의 주행 상태에 따라 YOLO 모델의 신뢰도(Confidence) 기준을 동적으로 변경했습니다.

정지/탐색(SEARCH): 임계값을 0.70으로 높여 오작동을 원천 차단했습니다.

접근/이동(TRACK): 임계값을 0.40으로 낮춰, 보행 진동이나 모션 블러가 발생해도 추적이 끊기지 않도록 주행 연속성을 극대화했습니다.

•IBVS (Image-Based Visual Servoing) 

제어복잡한 좌표 변환을 생략하고, 화면상 픽셀 좌표를 제어 입력으로 직접 사용하여 제어 안정성을 높였습니다.

조향(Yaw) 및 전진(X/Y) 제어: 인식된 계단의 중심 좌표가 화면 중앙(640px)에 위치하도록 회전 각속도를 제어하고, 계단 하단부가 목표선에 도달할 때까지 전진 속도를 제어합니다.

안전 정지선(Stop Line) 설정: 기구학적 역산을 통해 계단 전방 약 21.8cm 위치에 해당하는 화면 하단 650px 지점을 최적의 도착 지점(Stop Line)으로 설정하여, 충돌 없이 안전하게 등반을 준비할 수 있도록 마진을 확보했습니다 

•데드존 극복을 위한 동적 피치(Pitch) 

제어로봇이 계단에 근접할수록 계단이 카메라 화각 아래로 사라지는 데드존(Dead Zone) 문제를 해결하기 위해, 거리에 비례하여 로봇의 고개 각도를 제어했습니다.

선형 보간(Linear Interpolation) 적용: 화면상 계단의 위치(480px ~ 650px 구간)에 따라 목표 피치 각도를 0에서 최대 0.38 rad(약 21도)까지 실시간으로 계산하여 적용했습니다.

지수 이동 평균(EMA) 필터: 계산된 목표 피치 값에 가중치(Alpha=0.12)를 두어 과거 값과 혼합함으로써, 모터의 급격한 움직임을 방지하고 카메라 시야의 흔들림을 최소화하는 부드러운 모션을 구현했습니다.

•ROS 2 및 Unitree API 제어 통합

ROS 2 노드에서 계산된 명령을 로봇 하드웨어가 수행하도록 Unitree 내부 API 프로토콜을 연동했습니다.

Pose 모드 동시 제어: 초기화 단계에서 1028번 API(Pose 모드)를 강제 활성화하여, 이동(API 1008)과 동시에 몸통의 기울기(API 1007 - Euler Pitch)를 실시간으로 제어하는 복합 모션을 구현했습니다.

### 3. 트러블슈팅 및 고찰

3D 맵핑 노이즈 한계 극복: 초기에는 카메라와 라이다 좌표를 통합하여 3D 월드 좌표계로 변환하려 했으나, 로봇 보행 시 발생하는 강한 진동으로 인해 1m 앞의 거리 오차가 10cm 이상 발생하는 Extrinsic Calibration 난항을 겪었습니다. 

이에 복잡한 연산을 생략하고 2D 이미지 평면 제어(IBVS)로 전환하여 실시간성과 제어 신뢰성을 모두 확보했습니다.

향후 과제: 현재의 비전 기반 접근 제어를 넘어, 3개의 다리 관절 모터를 개별 제어하는 등반 알고리즘의 구체화, 또는 LiDAR를 활용한 3D Point Cloud 맵핑 융합을 통한 보행 고도화 또는 하드웨어적 구상(등반 시 지지팔을 활용)을 계획하고 있습니다.
